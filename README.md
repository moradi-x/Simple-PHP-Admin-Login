# Simple PHP Login System

A very simple login system using PHP sessions.  
Includes login page, admin page, and logout functionality.

## Features
- Login page
- Session authentication
- Admin dashboard (protected)
- Logout system
- Basic CSS styling

## Files
login.php      - Login form  
check.php      - Validates username & password  
admin.php      - Admin page (requires session)  
logout.php     - Ends session  
style.css      - Login page style  
style2.css     - Admin page style

## How to Run
1. Place project in htdocs (for XAMPP)  
2. Start Apache  
3. Open in browser:  
http://localhost/project/login.php
